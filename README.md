# EMAAR-Integrated-Mosque-Management-System
EMAAR is an integrated smart mosque management system designed to enhance operational efficiency, automate monitoring processes, and support digital transformation in mosque facilities across Saudi Arabia.  The system integrates IoT, Computer Vision, Automation, and Cloud Technologies to monitor mosque operations in real time.

## Problem Statement
Many mosques still rely on traditional manual workflows for maintenance, cleanliness monitoring, attendance verification, and utility tracking. This leads to delays, resource waste, and inefficient operational management.

## Solution
EMAAR provides a centralized smart management platform that combines:
- IoT-based monitoring
- Computer Vision
- Automated reporting
- QR complaint system
- Real-time dashboards

## Features
- Real-time water & electricity monitoring
- Cleanliness assessment using Computer Vision
- Imam & Muazzin attendance verification
- QR-based complaint submission
- Automated PDF reports
- Role-based dashboards
- Firebase cloud integration
